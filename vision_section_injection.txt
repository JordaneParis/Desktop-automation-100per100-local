## 🤖 Autonomous Desktop Agent (Vision)

The skill can act as an autonomous desktop agent that sees the screen and takes actions. This uses fast screen capture (`mss`) and OpenCV for real-time vision.

### Vision-Specific Actions

| Action | Parameters | Description |
|--------|------------|-------------|
| `screenshot_mss` | — | Fast screen capture (returns BGR image as base64 or numpy array in internal use) |
| `find_on_screen` | `template_path`, `threshold?` | Find template on screen, returns (x, y, confidence) or null |
| `click_position` | `x`, `y`, `duration?`, `button?` | Move mouse to (x,y) and click |
| `click_image` | `template_path`, `threshold?` | Find template and click it automatically |
| `read_text_ocr` | `lang?` | Capture full screen and return all OCR text |
| `read_text_region` | `x`, `y`, `width`, `height`, `lang?` | OCR on a specific screen region |

### Example: Autonomous Workflow

```javascript
// 1. Take a screenshot (optional, for debugging)
sessions_spawn({ task: 'screenshot_mss', label: 'desktop-automation-100per100-local' });

// 2. Find and click a button
sessions_spawn({
  task: 'click_image',
  params: { template_path: "C:/templates/next.png" },
  label: 'desktop-automation-100per100-local'
});

// 3. Wait a bit
time.sleep(1);

// 4. Read confirmation message
sessions_spawn({
  task: 'read_text_ocr',
  params: { lang: 'eng' },
  label: 'desktop-automation-100per100-local'
});
```
